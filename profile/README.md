# 📰 Tech-Pick ![Progress](https://img.shields.io/badge/Status-In_Progress-orange)
> **AI 기반 IT 뉴스 큐레이션 및 기술 Q&A 비서**
>
> 단순한 뉴스 나열을 넘어, 전 세계 IT 트렌드를 국가별로 파악하고 AI(Gemini)를 통해 실시간 기술 상담을 받을 수 있는 서비스입니다. AI 회사에서의 모델 활용 경험과 C2C 기업에서의 앱 개발 기술력을 조합하여 웹 기반의 고성능 서비스를 구축합니다.

### 🛠 Tech Stack
- **Frontend:** `React`, `Tailwind CSS` (Deployed on **Netlify**)
- **Backend:** `Java`, `Spring Boot`, `Spring Security (JWT)` (Deployed on **Cloudtype**)
- **Database:** `PostgreSQL` (Cloudtype Managed DB)
- **AI:** `Google Gemini API`

**[Key Features]**
- 최신 IT 뉴스 크롤링 및 나라별 제공
- Gemini API를 활용한 개발 기술 Q&A 챗봇 서비스
- JWT 기반의 안전한 회원가입 및 로그인 기능
